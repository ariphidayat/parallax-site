## Parallax Site

### Prerequisite
* Installed [Node.js] (https://nodejs.org) and [npm] (https://www.npmjs.com/)
* Installed [bower-cli] (http://bower.io/)

### Build and Run :

* Execute the following command :

        - execute `npm install`
        - execute `bower install`
        - execute `grunt`

* Call URL
        
        http://localhost:3000
